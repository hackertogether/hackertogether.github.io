フリマアラートの基本的な使い方、トラブル時の対応等をまとめました。
お問い合わせの前にこちらをご確認いただけると大変嬉しいです。

アプリのレビューで不具合をご報告いただいても詳細の確認ができませんので、お問い合わせよりご連絡ください。
お問い合わせで返信をお求めの場合は、必ず「monitorflea@yahoo.co.jp」からのメールを許可するようにご設定願います。
フリーメールで恐縮ですが必ず返信いたしますので、ドメインの許可設定をお願いいたします。

::: {.sub_ttl}
[◆]{.cube}どんなアプリなの？
:::

::: {.sub_cntnt"}
お目当ての商品を見つけてもすでに売り切れていませんか？
安い商品をゲットするために何度も検索して手間がかかっていませんか？
「フリマアラート」はそんなあなたに代わって出品された商品を監視＆通知します。
:::

::: {.sub_ttl}
[◆]{.cube}[商品の価格が実際と違う]{.alrt}
:::

::: {.sub_cntnt"}
フリマアラートでは出品時の商品ステータスを取得して通知しているため、通知後に商品詳細が変更されると、フリマアラート側の情報と異なってしまいます。
最近メルカリの仕様で10%以上値下げを行うと再度新着商品として扱われるため、意図的に価格を一旦下げてから上げるケースもあるようです。
またフリマアラートのような監視ツールで即買いされる方を対象とした詐欺行為である可能性も考えられますので、必ずご購入前にその時の情報をご確認いただけますようお願いいたします。
購入してしまった場合、基本的にキャンセルはできない仕組みとなっておりますのでトラブルを避けるためにも、くれぐれもご注意願います。
:::

::: {.sub_ttl}
[◆]{.cube}通知優先度、ブーストって何？
:::

::: {.sub_cntnt"}
フリマアラートではウォッチを順番に処理しているため、先頭と最後では通知速度の差が多少ございます。
ブーストボタンを押して広告動画を見ていただくことで通知の優先度を上げることができます。通知の優先度ランクはS～Fまでとなっており、ブースト直後は必ずSランクとなります。
しかしその後他の方がブーストすると徐々にランクが下がっていく仕様となっております。
優先度による差異は時間帯にもよりますが数秒程度ですので、今これから張り込みたい！というような時にご利用いただくと良いかと思います。
そのため、秒で争うような商品を狙っていない方には不要な機能かと思います。また広告動画の再生には通信量の負荷もございますのでご注意ください。
:::

::: {.sub_ttl}
[◆]{.cube}通知が来ない
:::

::: {.sub_cntnt"}
①フリマアラートはアプリにアクセスしないまま1週間経過するとウォッチを自動的に停止します。
停止する12時間前に一度通知でお知らせしますので、その際にアプリを起動していただければ継続して1週間動作します。
放置されたウォッチを処理し続けることはサービス全体の速度低下に繋がってしまいます。
毎日ご利用いただいているユーザー様のウォッチを優先的に処理するため、
放置判断の閾値を1週間とさせていただいております。
②未読の通知を50件以上溜めると、新着通知が停止します。あまり頻繁に通知を確認できない場合は通知の頻度を下げるためにウォッチ設定の見直しをお願いいたします。
③端末側の通知設定で「フリマアラート」の通知を許可してください。
iOS：「設定」→「通知」→「フリマアラート」の通知を許可に設定
Android：「設定」→「音声と通知」→「アプリの通知」→「フリマアラート」の「通知を表示」にチェックを入れる
Android：「設定」→「アプリ(アプリケーション管理)」→「フリマアラート」→「通知を表示」にチェックを入れる
※OSのバージョンによって設定方法が異なりますので上記の限りではございません。
④Wi-Fiのみで接続していませんか？LTEや3Gを使わずWi-Fiのみで接続している場合、設定によってスリープ状態への移行時にWi-Fi接続が切断されます。
⑤タスクキラー系アプリ、アンチウィルス系アプリが入っていませんか？
これらのアプリにより通知が制限されてしまう場合があります。
⑥上記でも解決しない場合はシステムトラブルの可能性もございますので、お問い合わせください。
お問い合わせの際は必ずご利用端末名、OSのバージョンをご記載願います。
:::

::: {.sub_ttl}
[◆]{.cube}機種変後通知が来ない(iOS)
:::

::: {.sub_cntnt"}
旧端末から新端末へのデータ移行をiTunes経由で行った場合、新旧両方が同IDとなるため正常に通知が届かなくなってしまう場合がございます。
一旦旧端末にて引継ぎコードとパスワードを設定し、新端末のフリマアラートを再インストール後、引継ぎを行っていただくと新端末にのみ通知が届くようになります。
改善しない場合はお手数ですがお問い合わせください。
:::

::: {.sub_ttl}
[◆]{.cube}通知が来ない、遅延する(Android)
:::

::: {.sub_cntnt"}
フリマアラートでは商品が出品されてから約1分以内の通知を実現できておりますが、
送信したプッシュ通知がご利用の端末に届かなかったり、遅延するケースが最近報告されております。
①スリープ時の消費電力を抑えるDozeモード(Android6.0以降をお使いの方)
スリープ時の消費電力を抑えるDozeモードで「フリマアラート」と「Googleフレームワーク」を「最適化しない」に設定してください。
こちらは「最適化する」となっている場合、スリープ後に30分～1時間程度放置すると通知が届かなくなるようです。
②Huawei端末をご利用の方、省電力モードの設定をご確認ください。
・「設定」→「詳細設定」→「バッテリーマネージャー」→「保護されたアプリ」→「フリマアラート」を保護
・「設定」→「詳細設定」→「バッテリーマネージャー」→「電源プラン」→「ウルトラ省電力」以外に設定
※上記以外にも独自の設定があるかもしれませんが、全ての端末情報を把握はしきれておりませんので解決に至らない場合もございます。
:::

::: {.sub_ttl}
[◆]{.cube}通知が時々来ない
:::

::: {.sub_cntnt"}
プッシュ通知の仕様上、ご利用端末の通信状況やメモリ使用状況によって通知が届かない場合もございます。
また各フリマアプリへのアクセスが集中する時間帯は稀に商品の取り込み漏れが発生することがあります。
調査することは可能ですので、お気づきの点がございましたらお問い合わせください。
:::

::: {.sub_ttl}
[◆]{.cube}通知に金額を表示したい
:::

::: {.sub_cntnt"}
デフォルトでは表示しない設定になっておりますので、
「その他」→「プッシュ通知の金額表示」より表示する設定に変更してください。
:::

::: {.sub_ttl}
[◆]{.cube}検索とウォッチは何が違うの？
:::

::: {.sub_cntnt"}
ウォッチは新着商品をお知らせするための機能です。迅速に新着通知を行うことを最優先としておりますので、直近の出品商品しか表示しません。
検索は複数のフリマアプリで出品されている商品を検索したい場合に便利です。直近の商品のみではなく過去の出品商品も検索することができます。
また検索に利用した設定をそのままウォッチに登録することも可能です。
:::

::: {.sub_ttl}
[◆]{.cube}検索機能について
:::

::: {.sub_cntnt"}
①各フリマアプリの検索機能に依存しておりますので、OR検索が利用できないなど、ウォッチ機能の設定とは仕様が異なります。そのためウォッチでマッチする商品と検索結果の商品は必ずしも一致しないことをご了承ください。
②通信量を抑えるため、ある程度検索結果件数を絞り込んでおります。全件を表示することはできません。
③検索対象のフリマアプリはウォッチの検索対象と同様です。その他→利用フリマ設定よりご設定ください。
④商品状態、送料を検索条件に指定できますが、商品リストには商品状態や送料が表示されませんのでご注意ください。
:::

::: {.sub_ttl}
[◆]{.cube}特定の出品者をウォッチしたい
:::

::: {.sub_cntnt"}
出品者ウォッチは出品者IDで確実に拾うため、一度フリマアラート内でウォッチしたい出品者の商品を表示させ、「Seller」ボタンを押す必要があります。
商品を表示させるのは以下の2つの方法になります。
1.ウォッチ機能から登録する場合
　ウォッチ機能にて過去に該当出品者の商品が通知されている場合、その商品の「Seller」ボタンを押します。
　◆フリマアラートを起動⇒該当するウォッチをタップ⇒商品一覧から該当商品の「Seller」ボタンを押す
2.検索機能から登録する場合
　ウォッチ内にお目当ての出品者の商品がない場合は検索機能を使ってその商品をヒットさせ、検索結果商品一覧から該当商品の「Seller」ボタンを押します。
　◆フリマアラートを起動⇒検索タブを押す⇒検索ワードを入力してお目当ての出品者の商品を検索⇒検索結果商品一覧から該当商品の「Seller」ボタンを押す

「Seller」ボタンを押した後は「この出品者の商品をウォッチする」を押すと出品者ウォッチを作成することができます。
ご不明な点がございましたらお問い合わせ願います。
:::

::: {.sub_ttl}
[◆]{.cube}特定の出品者を除外したい
:::

::: {.sub_cntnt"}
フリマアラートを起動⇒該当するウォッチをタップ⇒ウォッチの通知済商品一覧を表示させ、除外したい出品者の商品の「Seller」ボタンを押す⇒「この出品者の商品をアラートしない」を押すとその出品者を今後除外することができます。(10件まで)
誤って押してしまった場合はウォッチ設定の出品者除外より解除することが可能です。
ご不明な点がございましたらお問い合わせ願います。
:::

::: {.sub_ttl}
[◆]{.cube}ウォッチをONにできない
:::

::: {.sub_cntnt"}
フリマサービスへの負荷を軽減するため、通知をONにできるウォッチの数を制限しています。
負荷状況により登録可能数は変動しますのでご了承ください。
新しくウォッチをONにする場合は、今設定されている通知ONのウォッチをOFFにしてください。
またはOR検索を駆使することによって、ひとつのウォッチで複数商品をマッチさせることも可能です。
:::

::: {.sub_ttl}
[◆]{.cube}フリマアプリに飛べない
:::

::: {.sub_cntnt"}
商品リストからフリマアプリに飛ぶ際、広告が表示されてしまう場合は広告表示時に起動しているアプリをご確認ください。
起動中のアプリ確認方法は以下の通りです。
iOS：ホームボタンを二度押し
Android：ホームボタンを長押し
最近複数の中国系音楽再生アプリなどで、メルカリなどのアプリへのリンクが関連付けられてしまい、
リンクを踏んだ際に勝手に起動して広告を表示するため該当アプリへ飛べなくなることがあるようです。
フリマアラートではフリマアプリへ飛ぶタイミングで広告を表示することはありません。
広告収入を増やすための悪質なアプリも最近増えておりますのでご注意ください。
参考サイト：iOS端末で全画面広告が開きアプリが落ちる現象
:::

::: {.sub_ttl}
[◆]{.cube}ヤフオクの通知が来ない
:::

::: {.sub_cntnt"}
ヤフオクはフリマモードの商品のみ通知対象とさせていただいております。
また、出品数がかなり多いためstoreの出品商品と自動出品ツールによる出品商品も除外しておりますのでご了承ください。
:::

::: {.sub_ttl}
[◆]{.cube}商品一覧がフリマアプリと違う
:::

::: {.sub_cntnt"}
当アプリのウォッチでは独自の仕組みで検索を行っているため、本家アプリと100%は一致しません。
ですが除外検索を駆使して絞り込みを行うことで、お目当ての商品だけを通知させることは可能かと思います。
どうしても意図せぬ検索結果となる場合はお問い合わせよりご相談ください。
:::

::: {.sub_ttl}
[◆]{.cube}エラーが出て開けなくなった
:::

::: {.sub_cntnt"}
『予期せぬエラーが発生しました』と表示される場合、サーバー障害が発生している可能性がございます。
通常数分で復旧いたしますが、画面が真っ白なまま機能しない場合は一度アプリをタスクから消し、起動し直してください。
しばらく経過しても改善されない場合は大変お手数ですが、お問い合わせにてご報告ください。
可能な限り即時対応いたしますので、よろしくお願いいたします。
:::
